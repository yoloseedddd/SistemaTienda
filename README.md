**SistemaTienda** es una plataforma de comercio electrónico completa y funcional diseñada para simular la experiencia real de compra y venta online. Este proyecto abarca desde la selección de productos por parte del cliente hasta la gestión de inventario por parte del administrador, integrando una base de datos relacional robusta.
---
¿Qué puedes hacer en este proyecto?
Para el Cliente (Experiencia de Compra)
* **Catálogo Interactivo:** Navega por productos con imágenes y descripciones, y usa el **buscador inteligente** para encontrar lo que necesitas.
* **Carrito de Compras:** Agrega productos, ve el subtotal en tiempo real y gestiona tus ítems.
* **Cupones de Descuento:** Aplica códigos promocionales (ej: `PROMO2026`) para obtener rebajas.
* **Simulación de Pago:** Una pasarela de pago realista que valida tarjetas de crédito (Visa/Mastercard) y simula la conexión bancaria.
* **Historial de Pedidos:** Accede a tu perfil para ver el estado de tus compras pasadas y reimprimir tus boletas.
Para el Administrador (Panel de Control)
* **Dashboard Visual:** Gráficos estadísticos de ventas e ingresos en tiempo real.
* **Gestión de Inventario (CRUD):** Agrega, edita o elimina productos directamente desde la web sin tocar código.
* **Control de Stock:** El sistema descuenta automáticamente el stock tras cada venta aprobada.
 Tecnologías Utilizadas
* **Backend:** Python (Flask)
* **Base de Datos:** MySQL (Estructura relacional con transacciones ACID)
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (para lógica de carrito y pagos)
* **Control de Versiones:** Git & GitHub
